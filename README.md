# Security Advisory
This repository contains important security information regarding this FrostClient.

## General Information
* This FrostClient is a Minecraft 1.7.10 and 1.8.9 client. Our 1.7.10 version uses the original 2018 CheatBreaker with new features, modifications and bug fixes to keep it up to date, and the 1.8.9 version is a reskin of Offline Cheatbreaker.
* This FrostClient is by no means a virus, or ratted in any way. All allegations saying otherwise are false. We do not recommend downloading any versions of this FrostClient from anywhere except our [website](https://www.frostclient.ml) and [GitHub](https://github.com/FrostClient).
* This FrostClient is available to the public in a form of a version file.
* All our game files are obfuscated.
* We provide cosmetics (capes and wings) for free (Open cosmetics menu to select the cosmetics you want).
* Our only official GitHub repositories are available under this [GitHub organisation account](https://github.com/FrosClient).

CheatBreaker's Official Security Advisory can be found [here](https://github.com/CheatBreaker/Security-Advisory).
